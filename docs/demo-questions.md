## Expected Demo Behavior

- The answer should start with `结论`.
- The answer should include `理由` and `下一步`.
- The answer should show reference chips under the assistant message.
- Clicking a reference chip should highlight the matching knowledge card.
- The suggestion buttons should change after mode/season changes or after a question is answered.
